# Requirements

## Project Name
Employee Attrition Prediction System

## Problem Statement
Organizations lose valuable employees due to attrition. HR departments need a system that predicts whether an employee is likely to leave so they can take preventive actions.

## Objective
To build a machine learning-based application that predicts employee attrition using employee-related information.

## Functional Requirements
- User Login
- Employee Data Input
- Attrition Prediction
- Dashboard with Analytics
- Reports and Visualization

## Non-Functional Requirements
- User-friendly interface
- Fast prediction
- Secure system
- Responsive design
- Accurate prediction

## Target Users
- HR Managers
- HR Executives
- Company Management
