# User Stories

### Story 1
As an HR Manager, I want to predict employee attrition so that I can take preventive actions.

### Story 2
As an HR Executive, I want to enter employee details so that I can get prediction results.

### Story 3
As a Manager, I want to view analytics dashboards so that I can understand attrition trends.

### Story 4
As an Admin, I want to manage employee records so that the data remains updated.
